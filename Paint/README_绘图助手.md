# My-OpenSAS 绘图助手

该工具用于从 `Output_data` 中自动识别 Pushover、时程分析和 IDA 结果，通过点击操作生成、编辑并导出图片，不需要修改 Python 脚本。

## 启动

在项目根目录双击：

```text
启动绘图助手.bat
```

也可以在项目根目录运行：

```powershell
.\.venv\Scripts\python.exe .\Paint\paint_ui.py
```

## 第一步：生成基础图片

1. 点击“浏览”，选择包含各个工况文件夹的 `Output_data`。
2. 点击“扫描结果”。
3. 勾选需要绘制或比较的模型、温度工况。
4. 选择 Pushover、时程分析或 IDA 分析。
5. 选择图片类型和响应指标。
6. 点击“生成图片”。

可直接用于测试的样本目录：

```text
G:\Opensees-PFSDF-Temp-Slfecentring\Output_data
```

## 第二步：编辑图片

生成基础图片后，点击“下一步：编辑图片”。可以调整：

- 图片标题；
- 图例显示、位置和列数；
- 坐标轴、刻度和图例字号；
- 曲线宽度和数据点大小；
- 画布宽度和高度；
- X、Y 轴显示范围；
- 主网格显示。

修改参数后点击“应用调整”。点击“恢复推荐样式”可恢复 Times New Roman/宋体、25 pt 轴标签、18 pt 刻度、15 pt 图例和 2 pt 线宽。

## 保存图片

点击预览区右上角“保存图片…”，支持：

- PNG；
- PDF；
- SVG；
- TIFF。

默认导出分辨率为 300 DPI，可以在保存按钮左侧调整。图片使用紧凑边界导出。

## 当前支持的图片

### Pushover

- 屋顶位移角—基底剪力；
- 屋顶位移—基底剪力；
- 归一化能力曲线。

### 时程分析

- 楼层响应剖面；
- 响应箱线图；
- 单条记录楼层峰值；
- 层间位移角时程；
- 梁铰、柱铰和节点域楼层分布。

### IDA

- IDA 单条曲线及分位线；
- 易损性曲线和温度—易损性曲面；
- PSDM 概率需求模型；
- 损伤超越概率；
- IDA 计算次数检查。

## 常见提示

- 某个工况没有对应分析结果时会自动变灰，不参与绘图。
- 时程图需要原始 `MC8_TH_<水准>_data` 目录中的 `Disp*.out` 和 `Time.out`。
- CSV 中文表头即使存在乱码，工具仍按数值列和文件位置读取。
- 易损性曲面至少需要两个带温度编号的工况。
